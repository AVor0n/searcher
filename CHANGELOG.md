## [0.0.3] - 2025-03-20

- Добавлен переключатель для поиска по регулярным выражениям

- Добавлено кэширование содержимого файлов

### Fixed
- Исправлен поиск по именам файлов
- Исправлен исключающий поиск
- Оптимизирована производительность поиска
